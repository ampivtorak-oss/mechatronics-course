# ЛР2 — Проєктування мехатронної системи (МС)

## Коротко про роботу
Цей репозиторій містить матеріали до ЛР2 з дисципліни «Мехатронні системи»: структурні/функціональні схеми МС (Draw.io), їх експорт для перегляду в GitHub, а також допоміжні текстові матеріали. Креслення, скріншоти та логи не включені.

## Вміст репозиторію
- `drawio/` — вихідні файли схем у Draw.io (`*.drawio`).
- `schemes_exports/` — експорт схем у форматах для перегляду (PNG/SVG/PDF).
- `docs/` — текстові матеріали (опис системи, специфікації, пояснення).
- `sources.md` — перелік джерел.
