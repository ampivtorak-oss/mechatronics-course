# Mini-reports (Міні-реферати)

Папка містить короткі роботи з курсу «Мехатронні системи». Кожен документ є окремим міні-рефератом, виконаним за завданнями з електронного курсу, з акцентом на: вибір/опис об’єкта, склад МС, структурну (функціональну) схему, інформаційні потоки та (де потрібно) спрощений математичний опис.

## Перелік робіт

### 1) Міні-реферат «Мехатронні системи навколо нас»
Коротко розкрито сутність мехатронних систем (інтеграція механіки, електроніки, сенсорики та керування). Як приклад розглянуто робот-пилосос: визначено призначення, межі системи, основні підсистеми та взаємодію елементів. Додано функціональну схему.

Файл: `Міні-рефера Мехатронні системи навколо нас.docx`

### 2) Опис виконавчих елементів МС (робот-пилосос)
Описано виконавчу частину робота-пилососа: приводи руху, привід щіток, турбіна/вентилятор (та додаткові виконавчі вузли за потреби). Наведено узагальнений схемний і спрощений математичний опис електропривода та приклади моделей для типових виконавчих елементів. Вказано сигнали керування і зворотного зв’язку.

Файл: `Опис виконавчих елементів МС робот пилосос.docx`

### 3) Структура МС підвісу БПЛА (класифікація, модель, структурна схема)
Обрано двоступеневий підвіс БПЛА (Pitch/Roll) як мехатронну систему стабілізації. Виконано класифікацію, описано склад апаратної частини, наведено спрощений математичний опис однієї осі та структурну схему контуру керування (регулятор–привід–об’єкт–датчик–зворотний зв’язок). Схема дублюється для другої осі.

Файл: `структура МС підвіс БПЛА.docx`

### 4) Опис ДІ (інформаційного забезпечення) МС підвісу БПЛА
Описано датчиково-інформаційну частину підвісу: IMU (гіроскопи/акселерометри), можливі датчики положення (енкодери/датчики Холла), обробку даних (фільтрація, оцінювання орієнтації), інформаційні потоки та вимоги до частоти оновлення/затримок. Додано блок-схему «сенсори → обробка → регулятор → силова частина» та зазначено типові механізми діагностики/безпеки.

Файл: `Реферат Опис ДІ МС підвіс БПЛА.docx`
