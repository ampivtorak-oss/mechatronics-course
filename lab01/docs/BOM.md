## 7) Очікувані технічні характеристики (для звіту ЛР-1)
- Ступені свободи: **2 осі (Pitch/Roll)**
- Тип приводу: **Direct-drive BLDC (gimbal motors)**
- Режими: **стабілізація + керування кутом від FC**
- Діапазон кутів (ціль): **Pitch ±45°, Roll ±45°** (фіналізація після CAD, щоб уникнути зачепів)
- Керування: **UART/RC**, Trigger/Sync — окремий канал
- Живлення: **від бортової шини через DC-DC/UBEC**, рекомендована фільтрація від EMI
